Install ImageMagick - CentOS7, CentOS 8

. Install ImageMagick on CentOS. 
.
sudo yum -y install ImageMagick
sudo yum -y install ImageMagick-devel
ImageMagic is ImageMagic, which allows you to use ImageMagick's convert command. ImageMagick-devel is a library for ImageMagic, which is written in C and is required to use ImageMagick's features from other programming languages such as Perl, PHP and Ruby.
